# Solace Cafe — Cafe Website

A multi-page cafe website for **Solace Cafe**, built with HTML, CSS, and JavaScript.  
*Where coffee meets comfort.*

---

## Features

- **Home** — Hero section, about us, menu preview, contact form, and footer with opening hours and location
- **Menu** — Category tabs (Pasta, Asian, Pastries, Sides, Drinks) with Coffee / Non-Coffee sub-tabs, quantity selectors, and Add to Cart
- **Cart** — View items, change quantity, choose delivery or dine-in, tax and delivery fee, proceed to checkout
- **Checkout** — Complete order flow from cart
- **Reservation** — Date and time selection for table booking
- **Login / Sign up** — Sign in and registration forms
- **Our team** — Team page
- **Contact** — Feedback form and contact details (email, phone)
- **Scroll to top** — Button to scroll back to the top of the page

## How to run

1. Clone or download the **Cafe-website-assignment** folder (the folder that contains `index.html`).
2. Open `index.html` in a browser (double-click or “Open with” your browser).
3. Or use a local server:
   - **VS Code:** Install “Live Server” and right-click `index.html` → “Open with Live Server”
   - **Python:** From the project root run `python -m http.server 8000`, then go to `http://localhost:8000`

No build step or extra dependencies are required.
